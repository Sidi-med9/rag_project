Category: Security and Accounts
Title: Incident Response Protocol for Ransomware Infection Outbreaks
ID: SEC_03
Symptoms: Files on a workstation change extensions unexpectedly to malicious encryptions, and a ransom note text file pops up.
Resolution:
1. Immediately pull out the physical Ethernet cable from the target machine and cut off all Wi-Fi connections.
2. Do not attempt to reboot the workstation as this could accelerate encryption malware processes.
3. Report the incident to the central Cyber Security Team, noting down the exact machine name and IP address.
4. Take a clear photograph of the ransom screen notes for intelligence reporting.
5. Use the network management switch console to isolate the infected network port or VLAN block.
6. Initiate enterprise-wide backup validation sequences to check core storage snapshots before restoration.
