Category: Operating Systems
Title: Adjusting Virtual Memory Paging Files for Stability
ID: OS_10
Symptoms: Out of Memory error messages appear during heavy workflows, crashing resource-intensive processes.
Resolution:
1. Open Advanced System Settings and click the Settings button under the Performance tab.
2. Move to the Advanced tab and select Change under the Virtual Memory heading.
3. Uncheck 'Automatically manage paging file size for all drives'.
4. Select the primary OS drive, choose 'Custom size', and set initial size to 1.5 times system RAM.
5. Set maximum size limit to 3 times system RAM to safeguard stability.
6. Save the parameters and execute a machine restart.
