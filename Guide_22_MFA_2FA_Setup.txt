Category: Security and Accounts
Title: Setting Up and Resetting Multi-Factor Authentication (MFA)
ID: SEC_02
Symptoms: Users get blocked from enterprise email portals because they lost access to their registered mobile authenticator device.
Resolution:
1. Log into the centralized tenant administration control console as a Security Administrator.
2. Locate the user identity record, open security settings, and select 'Require re-register multifactor authentication'.
3. Instruct the user to sign in from a clean workstation web browser session.
4. Have the user scan the newly generated QR code configuration using the official corporate authenticator mobile application.
5. Generate a collection of emergency backup alphanumeric recovery bypass keys and store them securely offline.
