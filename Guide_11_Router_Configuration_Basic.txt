Category: Networking and Communications
Title: Configuring Local Office Router Security Protocols
ID: NET_01
Symptoms: Deploying a new access point or updating weak local wireless encryption methods.
Resolution:
1. Connect a workstation directly to the router hardware via an Ethernet line.
2. Access the gateway management portal by typing the default IP (e.g., 192.168.1.1) into a web browser.
3. Supply default administrator credentials, then immediately modify them to a custom password.
4. Alter the network Service Set Identifier (SSID) to mask the hardware vendor name.
5. Change wireless security encryption standard to WPA3 or WPA2-Enterprise.
6. Save changes and reboot the router.
