Category: Operating Systems
Title: Overcoming Stuck Windows Update Sequences
ID: OS_06
Symptoms: Windows Update hangs at a specific percentage or returns error codes like 0x80070002.
Resolution:
1. Open the Services console by typing services.msc in the run dialogue.
2. Stop the Windows Update service and the Background Intelligent Transfer Service (BITS).
3. Navigate to C:\Windows\SoftwareDistribution\Download and delete all files inside to clear corrupt update state.
4. Return to the Services console and restart both Windows Update and BITS.
5. Run the built-in Windows Update Troubleshooter to clear cached policies.
